📘 About This Project

This is a beginner-friendly DevOps learning project built using Flask and HTML UI.

🚀 Project Purpose
To learn web development basics, Git workflow, and DevOps fundamentals step by step.
💾 Git & GitHub Usage
This project is managed using Git for version control and stored on GitHub for collaboration and backup.
📂 Project View
- Flask backend handles routing
- HTML pages for UI
- Git tracks all changes
- GitHub stores full project history
